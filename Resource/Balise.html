Les differentes balises HTML

Le texte--------------------------------------------------------

Pour ecrire du texte en html nous utilisont la balise "p" "/p"

Nous pouvont aussi utiliser la balise "h1" qui vas juste que la balise "h6"

elle permet de mettre en gras est de plus ou moins agrandire le texte.

La balise "strong" "/strong" permet de mettre le texte en gras.


Les listes------------------------------------------------------

Pour crée une liste en html nous avont besoins de la balise "ul" "/ul"

pour remplir cette liste d'éléments on utilise les balises "li" "/li" pour définir un élément de la liste


Les Tableaux-------------------------------------------------------

Pour créer un tableaux html nous utiliseront la balise "table" "/table".

Ensuite nous utilisont la balise "tr" "/tr".

Aprés nous povont utiliser la balise "th" "/th" pour lister les colones.

Puis nous pouvont utiliser "td" "/td" pour faire les lignes.
