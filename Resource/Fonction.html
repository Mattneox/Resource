Les bases des Fonctions

Création d'une fonction----------------------------------------------------

Une fonction sert à écrit du html dans une fonction est a pouvoir le reutiliser sur plusieurs autres fichier php

Par exemple :

  • (function texteHtml()) ont déclare une fonction
  • (EOT)Ont utilise la syntaxe Heredoc
  • Ont recopier le texte html de "!doctype" jusque "body"
  • Puis ont ferme le texte html et le EOT

Cette méthode est pour ecrire du HTML il exsiste plus de maniere d'utiliser une fonction


Une fonction avec argument-------------------------------------------

Avec la fonction "function texteHtml()" il nous suffit d écrire le nom de argument "function texteHtml($auteur)"

Puis ont ecrit l'argument la ou on veut qui apparaise "$auteur © 2022"

Et pour finir on ecrit se qu'il vas y avoir dans l'argument dans un autre fichier php texteHtml("PERIN Matteo");


Une fonction avec argument facultatifs-------------------------------------------

L'on peut incrementer un fonction directement "function texteHtml ($auteur = PERIN Matteo)"

Cette valeur sera une valeur par defaut elle sera pris quand il n'y a pas de valeur dans l'autre fichier php "texteHtml(" ");"

Si il y a une valeur dans l'autre fichier php, ces cette valeur qui vas etre prioritaires.


Variables locales et globales-----------------------------------------------------

Si ont utilise une variable dans une fonction ont ne pourras pas l'utiliser en dehord de la fonction.

Il exsiste 3 methodes pour pouvoir utiliser une variable en dehord de la fonction:

  • 1) Retourner la valeur au programme appelant

    Cette premiére méthode est la plus utiliser en entreprise.

    Elle consiste à déclarer la variable dans la fonction est de marquer "return $nomVariable".

    Aprés l'on peut écrire "echo $nomVariable" ce sera la variable saisi dans la fonction qui sera récuperer.


  • 2) Utiliser la variable du programme appelant comme argument de la fonction

    Cette deuxieme méthode est à bannir car elle peut engranger des problémes.

    Elle consiste à donner le nom de la variable comme argument "function test($nomVariable)".

    Mais le risque de modification involontaire est important.


  • 3) Déclarer la variable dans la fonction comme étant globale

    Cette variable est déconseille car il peut y avoir les même probleme que pour la deuxieme méthode

    Elle consiste à écrire "global $nomVariable"

    Mais comme la deuxieme méthode le risque de modification involontaire est important.
