Variables et Constantes

Les variables en php------------------------------------

Les variables se definise par le caractére $ "$nomVariable"

Exemple : $age = 10;


Les constantes en php-------------------------------------

Les constantes se definise avec la commande define("nom de la constante, valeur")

Exemple : define('majeur',18);


La concaténation de texte en php---------------------------

Pour concaténer du texte le symbole est le point

Exemple : 'La'.'concatenation s effectuer avec'.' un point'


La concaténation de variables en php--------------------------------

concaténer des variables le symbole est le meme que pour le texte(. le point)


Quotes ou Guillemets ?---------------------------------------

Les guillements peuvent séparer d'elle même les variables des chaines de caractéres.

Les quotes ne séparer pas les variables il faut faire une concatenations.

Pour les requete SQL il sera preferable d'utiliser les guillemets pour ne pas être obliger de faire une concatenations.

Mais l'utilisation de quotes est le plus souvent utiliser et moins problematiques.


Les tableaux

Variables tableaux------------------------------------

Pour utiliser une variable de type tableaux ont utilise des [].

Exemple : semaine['lundi','mardi','mercredi','jeudi','vendredi','samedi','dimanche'];

Exemple : semaine[1,2,3,4,5,6,7];

Pour afficher tous le contenu du tableaux ont utilise "var-dump($nomTableaux);"

sinon ont utilise "echo $nomTableaux [];" pour afficher une valeur du tableaux.

Pour affecter de nouvelle valeur dans le tableaux ont utilise "$nomTableaux [] = 'nouvelleValeur'".

Pour supprimer une valeur du tableaux ont utilise "unset($nomTableaux[Valeur]"


Tableaux associatifs----------------------------------------

Pour associer dans un tableaux a la fois les valeur et le nom des valeur ont ecrit :

$nomTableaux [ 'nomValeur0' => valeur0, 'nomValeur1' => valeur1, 'nomValeur2' => valeur2,];

Pour ajouter une nouvelle valeur avec un nom on utilise "$nomTableaux['nomValeur']=valeur);"

Pour supprimer une valeur du tableaux ont utilise "unset($nomTableaux['nomValeur'])"


Tableaux multidimensionnels-----------------------------------

Pour affecter plusieurs tableaux dans un tableaux ont ecrit :

$nomTableuxPrincipal = ['nomTableux1' = ['nomValeur1' = valeur1, 'nomValeur2' = valeur2], 'nomTableux2'= nomValeur1' = valeur1, 'nomValeur2' = valeur2]]


Les opérateurs

Les opérateurs d'affectations---------------------------------

"Pour effectuer des opération nous pouvont utiliser les caractere suivant

"valeurSasie" += "valeur" (pour rajouter la valeur saisie)

"valeurSasie" -= "valeur" (pour enlever la valeur saisie)

"valeurSasie" /= "valeur" (pour diviser par la valeur saisie)

"valeurSasie" *= "valeur" (pour multiplier par la valeur saisie)


Les opérateurs de comparaison------------------------------------

Pour comparais des valeur nous pouvont utiliser les caractere suivant

var_dump(valeur1 == valeur2); (pour voir si les valeurs sont égals)

var_dump(valeur1 === valeur2); (pour comparer si ces le meme type (int,string,...))

var_dump(valeur1 != valeur2); (pour si les valeur sont differentes)

var_dump(valeur1 !== valeur2); (pour voir si les valeurs sont strictements differentes)

var_dump(valeur1 < valeur2); (pour voir si la valeur1 est plus petit que la valeur2)

var_dump(valeur1 > valeur2); (pour voir si la valeur1 est plus grand que la valeur2)

var_dump(valeur1 <= valeur2); (pour voir si la valeur1 est plus petit ou égale a la valeur2)

var_dump(valeur1 >= valeur2); (pour voir si la valeur1 est plus grand ou égale a la valeur2)


Les opérateurs logiques---------------------------

"&&" si il y a une fois "false" le resultat seras "false"

"||" si il y a une fois "true" le resultat seras "true"

"XOR" si il y a une fois "true" le resultat seras "true"
